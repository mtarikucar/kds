# Products (Add-ons)

**Menu path:** Side menu → Marketplace &nbsp;•&nbsp; **URL:** `/admin/marketplace`
**Access:** ADMIN, MANAGER • **Plan:** All plans

Every item you can buy in the Marketplace is documented below, one by one. Each
card lists: **what it does**, **price**, **recurring vs one-time**, and any
**dependency**.

<Callout type="info">
  The purchase path is the same for every add-on: add it to the cart in the
  Marketplace → pay via PayTR → the entitlement activates automatically. For the
  step-by-step walkthrough see [Purchase Flow](/en/marketplace/purchase-flow).
  Prices are TRY, KDV-inclusive.
</Callout>
