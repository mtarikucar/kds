# Feature matrix

The tables below compare what each of the three sellable plans (**Basic / Pro / Business**) includes and their usage limits. Use this page to decide which plan fits you best.

## Features

| Feature | Basic | Pro | Business |
| --- | :---: | :---: | :---: |
| POS (sales screen) access | ✓ | ✓ | ✓ |
| KDS (kitchen display) integration | ✓ | ✓ | ✓ |
| Inventory tracking | ✓ | ✓ | ✓ |
| Advanced reports | ✗ | ✓ | ✓ |
| Multi-location | ✗ | ✓ | ✓ |
| Custom branding | ✗ | ✓ | ✓ |
| Priority support | ✗ | ✓ | ✓ |
| Reservation system | ✗ | ✓ | ✓ |
| Personnel management | ✗ | ✓ | ✓ |
| Delivery integration | ✗ | ✓ | ✓ |
| API access | ✗ | ✗ | ✓ |
| External display (Partner Display) | ✗ | ✗ | ✓ |

<Callout type="info">
  **API access** and **external display (Partner Display)** are exclusive to the Business plan. These two capabilities cover third-party integrations and remote/desktop screen sessions.
</Callout>

## Limits

The infinity sign (**∞**) means that limit is **unlimited**.

| Limit | Basic | Pro | Business |
| --- | :---: | :---: | :---: |
| Max users | 5 | 15 | ∞ |
| Max tables | 20 | 50 | ∞ |
| Max branches | 1 | 3 | ∞ |
| Max products | 100 | 500 | ∞ |
| Max categories | 20 | 50 | ∞ |
| Max monthly orders | 500 | 2,000 | ∞ |

## The Trial plan

The **Trial** plan that every new business starts on has **all of Business's features and unlimited limits** for 7 days. Because it isn't a plan you buy, it isn't listed on the plan-selection screen and is excluded from the pricing comparison. For details, see [Trial & billing](/en/plans/trial-and-billing).

## Which features do I have right now?

You can see your business's currently active features in the panel. The access you see is your **actual access**, combined from your plan's features, any add-ons you've added from the Marketplace, and any capabilities granted specifically to you — it reflects what you can actually use, not just the plan's default list.

<Callout type="info">
  If you see a feature is off on your plan, you can unlock it by **upgrading your plan** or with a **Marketplace add-on**. See [Choosing & upgrading](/en/plans/choosing-and-upgrading) and [Marketplace](/en/marketplace).
</Callout>
